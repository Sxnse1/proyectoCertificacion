# Comandos para configurar Heroku

## 1. Configurar SESSION_SECRET en Heroku
heroku config:set SESSION_SECRET=$(node -e "console.log(require('crypto').randomBytes(64).toString('hex'))")

## 2. Ver todas las variables de entorno en Heroku
heroku config

## 3. Configurar NODE_ENV en Heroku (si no está configurado)
heroku config:set NODE_ENV=production

## 4. Ver logs en tiempo real
heroku logs --tail

## 5. Reiniciar la aplicación después de cambios
heroku restart

## 6. Verificar que las variables están configuradas correctamente
heroku config:get SESSION_SECRET
heroku config:get NODE_ENV

## Nota: También puedes usar la web de Heroku para configurar variables de entorno:
# https://dashboard.heroku.com/apps/tu-app-name/settings
